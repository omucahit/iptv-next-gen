# IPTV Next Gen

IPTV Next Gen is a modern, high-performance IPTV player for Windows designed for speed and compatibility. It handles large playlists efficiently and supports a wide range of video formats.

## Installation and Usage
No installation is required. This application is portable.

1.  Download the **release zip file** (e.g., `iptv_next_release.zip`).
2.  **Extract** the contents of the zip file to a folder on your computer.
3.  Open the extracted folder and double-click `iptv_next.exe` to launch the application.

*Note: You do not need to install any external players or codecs. Everything needed is included in the package.*

## Key Features

### Advanced Playlist Management
-   **Dual Source Support**: Load M3U playlists directly from a URL or select a local `.m3u` file from your computer.
-   **Smart Persistence**: The application remembers your last used playlist and settings, so you don't have to re-enter them every time.
-   **performance Caching**: Playlist data is cached locally to ensure instant startup times, even with large lists.

### Intelligent Filtering and Search
-   **Country Filtering**: Automatically detects country codes in your playlist groups (e.g., TR, DE, US) and lets you filter the entire channel list by country.
-   **Category Organization**: Easily switch between Live TV, Movies, and Series using the quick access tabs.
-   **Global Search**: Instantly search across all channels, movies, and series. The search checks both channel names and program titles for accurate results.

### Modern Video Playback
-   **High-Quality Player**: Built on top of MediaKit, ensuring smooth playback for HD and 4K streams.
-   **Extensive Codec Support**: Native support for modern video codecs including HEVC (H.265), eliminating "black screen" issues common in older players.
-   **Adaptive Streaming**: Handles HLS (m3u8) streams efficiently with automatic quality adjustment.

## How to Configure
1.  Click the **Settings** (gear icon) in the top-left corner.
2.  Choose your source:
    -   **M3U URL**: Enter the link provided by your IPTV provider. The app will verify and update this automatically.
    -   **Local File**: Click "Pick" to select a file stored on your computer.
3.  Click "Save & Update" to load your channels.

## Requirements
-   Windows 10 or Windows 11 (64-bit)
