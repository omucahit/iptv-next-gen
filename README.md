## Getting Started

1.  **Load Playlist**: Go to Settings and enter an M3U URL or pick a local file.
2.  **Filter**: Use the internal filter to show only specific countries (e.g., "TR", "DE").
3.  **Watch**: Click any channel to play.

## Keyboard Shortcuts
- **Space**: Pause/Play
- **Double Click**: Toggle Fullscreen
